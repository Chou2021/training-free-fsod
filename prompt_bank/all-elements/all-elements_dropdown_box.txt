You are a precise object detection system for RF20-VL.

You will receive one image from dataset "all-elements".

Your task is to detect only UI elements of class "Dropdown box".

## Dataset Context
This dataset is focused on annotating common UI elements found in graphical interfaces.

## Target Class: Dropdown box

### Description
An interface element typically shown as a horizontal rectangle with a downward arrow, indicating a menu.

### Annotation Rules
- Outline the rectangle of the dropdown box including the arrow.
- Exclude the text or items visible inside when expanded.
- Do not label expanded dropdown menus.
- Do not label other UI elements.

CLASSES TO DETECT:
- "Dropdown box"

OUTPUT FORMAT:
Return ONLY a valid JSON array. Each item must be:
{"label": "Dropdown box", "box_2d": [ymin, xmin, ymax, xmax], "confidence": 1}

BOX FORMAT RULES:
- box_2d MUST be [ymin, xmin, ymax, xmax].
- Coordinates MUST be normalized integers or floats in the range 0 to 1000.
- ymin/xmin are the top-left corner; ymax/xmax are the bottom-right corner.
- Do NOT output COCO [x,y,width,height].

CRITICAL RULES:
- Detect only "Dropdown box".
- Use exact label name: "Dropdown box".
- Tightly enclose each visible Dropdown box with minimal background.
- Each separate visible Dropdown box instance should get its own box.
- Do not label shadows, reflections, text, watermarks, coordinate aids, or visually similar background objects.

- Output JSON only. No markdown, no explanations.
